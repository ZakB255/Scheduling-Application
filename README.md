# Appointment Scheduling Application

A desktop scheduling application built with **C#**, **.NET**, **Windows Forms**, and **MySQL**. The application allows users to manage customers and appointments while storing data in a relational database.

> **Note:** This project was completed as part of the Bachelor of Science in Software Engineering program at Western Governors University and is shared to demonstrate database integration, object-oriented programming, and desktop application development skills.

---

## Features

- User login authentication
- Customer management
- Appointment management
- Create, edit, and delete appointments
- MySQL database integration
- Reports dashboard
- Data validation
- Search functionality

## Installation

1. Clone the repository.

```bash
git clone https://github.com/ZakB255/Appointment-Scheduling-Application.git
```

2. Open the solution in Visual Studio.

3. Configure the MySQL connection string if needed.

4. Build and run the application.
